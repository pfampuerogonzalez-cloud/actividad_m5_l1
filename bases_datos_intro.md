# 1. ROLL DE UNA BASE DE DATOS

### EXPLICA CON TUS PALABRAS CUÁL ES EL ROL DE UNA BASE DE DATOS RELACIONAL DENTRO DE UNA EMPRESA O ORGANIZACIÓN.

- R. EL ROL DE UNA BASE DE DATOS RELACIONAL ES CENTRALIZAR, ESTRUCTURAR Y PROTEGER INFORMACION PARA QUE LOS DATOS DE DIFERENTES DEPARTEMENTOS SE CONECTEN ENTRE SÍ.

### MENCIONA 3 EJEMPLOS CONCRETOS DE USO.

- R. UN SISTEMA DE VENTAS Y FACUTARICÓN, LA GESTION DE USUARIOS, ROLES Y PERMISOS Y UN CONTROL DE MATRICULAS Y NOTAS.

# 2. CARACTERÍSTICAS DE UN RDBMS

### DEFINE QUÉ ES UN RDBMS Y NOMBRA AL MENOS 3 CARACTERÍSTICAS QUE LO DIFERENCIAN DE OTROS TIPOS DE SISTEMAS DE ALMACENAMIENTO.

- R. ES UN SOFTWARE QUE PERMITE CREAR, ACTUALIZAR, ADMINISTRAR Y CONSULTA A UNA BASE DE DATOS RELACIONAL LA CUAL USA EL LENGUAJE SQL ESTANDAR, PROPIEDADES ACID ESTRICTAS Y INTEGRIDAD REFERENCIAL POR LLAVES, LAS CUALES LO DIFERENCIAN DE OTROS TIPOS DE SISTEMAS DE ALMACENAMIENTO.

### MENCIONA 3 RDBMS AMPLIAMENTE USADOS EN LA INDUSTRIAL Y EN QUE CONTEXTOS SE SUELEN UTILIZAR.

- R. POSTGESQL, MYSQL Y SQLITE

# 3. HERRAMIENTAS Y OBJETOS

### QUÉ HERRAMIENTAS GRAFICAS Y DE LÍNEA DE COMANDOS PUEDE USAR PARA CONSULTAR UNA BASE DE DATOS? MENCIONA AL MENOS DOS.

- R. HERRAMIENTAS GRÁFICAS: PGADMIN-MONGODB COMPASS
-    HERRAMIENTAS DE LÍNEA DE COMANDOS: MYSQL-SQLITE3

### DESCRIBE BREVEMENTE QUÉ FUNCION CUMPLE CADA UNOS DE ESTOS OBJETOS DENTRO DE UNA BASE DE DATOS.

- TABLA: ESTRUCTURA PRINCIPAL DONDE SE ALMACENAN FÍSICAMENTE LOS DATOS ORGANIZADOS EN FILAS Y COLUMNAS.
- VISTA: TABLA VIRTUAL BASADA EN EL RESULTADO DE UNA CONSULTA SQL PREVIA.
- INDICE: FUNCIONA COMO EL INDICE DE UN LIBRO, ESTRUCTURA QUE ACELERA LA VELOCIDAD DE BUSQUEDA Y RECUPERACION DE DATOS DE UNA TABLA.
- LLAVE PRIMARIA: CAMPO O CONJUNTO DE CAMPOS QUE IDENTIFICA DE FORMA UNICA A CADA REGISTRO DE UNA TABLA.
- LLAVE FORÁNEA:LLAVE FORÁNEA: COLUMNA QUE CONECTA UNA TABLA CON OTRA. ASEGURA QUE LOS DATOS RELACIONADOS SON VALIDOS Y QUE EXISTAN EN LA TABLA DE ORIGEN.
